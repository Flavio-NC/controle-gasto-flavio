COMO ATUALIZAR NO GITHUB

1. Entre no repositório controle-gasto-flavio.
2. Envie/substitua estes arquivos na tela principal do repositório:
   - index.html
   - manifest.json
   - service-worker.js
   - icon-192.png
   - icon-512.png
3. Clique em Commit changes.
4. Aguarde o GitHub Pages publicar.
5. No celular, abra o link do app no Chrome.

IMPORTANTE
Se o app ainda aparecer antigo, abra o link e atualize a página duas vezes, ou limpe o cache/site data do Chrome para esse site.
