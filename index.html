Instalar no Android

Se você abriu o arquivo direto da pasta Downloads, o Chrome pode não mostrar “Adicionar à tela inicial”. Para instalar como app, o ideal é usar a versão do ZIP em um endereço HTTPS, como GitHub Pages, Netlify ou outro hospedador. Mesmo assim, o botão Instalar acima mostra as instruções quando o celular não permite instalar direto.
